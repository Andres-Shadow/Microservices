# Repositorio de Microservicios

En este repositorio se almancenaran los diferentes talleres proporcionados en la materia Arquitectura Orientada a Microservicios. El avance los mismo se hará de forma periodica a médida que se avance con el desarrollo.

Cuenta con dos carpetas:

- Notes: contiene apuntes de clase que son tomados durantes las explicaciones de las clases teóricas.
- Talleres: contiene el desarrollo de los diferentes talleres proporcionados en la asignatura.
